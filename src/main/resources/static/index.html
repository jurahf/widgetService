Widgets web API

Create new widget: PUT /widget

Body parameters: x, y, z_index, width, height

Get by id: GET /widget/{id}

Get list of widgets (by pages): GET /widget

Request parameters: limit (default 10), page (default 1)

Update widget: POST /widget/{id}

Body parameters: x, y, z_index, width, height

Delete widget: DELETE /widget/{id}
